# 👻 Horror Maze Online

恐怖迷宮逃脫遊戲 - 一個充滿挑戰和恐怖氛圍的多人在線遊戲

## 🎮 遊戲特性

- **獨特的遊戲玩法**
  - 🔦 手電筒照明系統 - 營造恐怖氛圍
  - 👻 智能 AI 鬼 - 使用改進的追逐算法
  - 🗝️ 鑰匙收集機制 - 需要收集所有鑰匙才能逃脫
  - ⏱️ 時間限制 - 增加緊張感

- **難度系統**
  - 🟢 簡單 - 15 分鐘，3 把鑰匙
  - 🟡 正常 - 10 分鐘，5 把鑰匙
  - 🔴 困難 - 5 分鐘，7 把鑰匙
  - 💀 地獄 - 3 分鐘，10 把鑰匙

- **移動平台支持**
  - 📱 虛擬搖桿控制
  - ⌨️ 鍵盤控制
  - 🎮 手機按鈕界面

- **高級功能**
  - 🔊 動態音效系統
  - 🎨 粒子效果與動畫
  - 🏆 高分排行榜（本地存儲）
  - 🌐 多人在線支持（WebSocket）

## 📋 系統要求

- 現代瀏覽器（Chrome, Firefox, Safari, Edge）
- JavaScript 啟用
- WebGL 支持（用於渲染優化）

## 🚀 快速開始

### 單人遊戲
1. 打開 `index.html`
2. 點擊「單人遊戲」
3. 選擇難度
4. 使用鍵盤或搖桿移動

### 鍵盤控制
- **移動**: W/A/S/D 或 方向鍵
- **奔跑**: Shift (需要體力)
- **手電筒**: F

### 目標
1. 收集所有金色鑰匙 🔑
2. 逃離綠色出口 🚪
3. 避免被紅色的鬼抓到 👻

## 🎯 遊戲機制

### 玩家系統
- 體力條: 奔跑消耗體力，靜止時恢復
- 視覺化指示: 眼睛方向顯示玩家朝向
- 平滑移動: 使用離屏 canvas 優化渲染

### 鬼 AI
- 追逐範圍內主動追捕玩家
- 範圍外隨機遊走
- 改進的尋路算法
- 被動技能：避開牆壁

### 照明系統
- 手電筒提供有限的照明範圍
- 關閉手電筒完全黑暗
- 鬼只在照明範圍內可見

### 音效系統
- 使用 Web Audio API
- 收集鑰匙音效
- 鬼接近警告
- 勝利/失敗音效

## 🌐 多人遊戲

### WebSocket 伺服器需求
- Node.js + WebSocket 庫
- 支持房間/大廳系統
- 玩家狀態同步

### 連接方式
```javascript
NetworkManager.init('玩家名稱', 'ws://localhost:8080');
```

## 📊 高分系統

### 本地存儲
- 自動保存前 10 名高分
- localStorage 格式: `horror_maze_scores`
- 包含玩家名、分數、難度、時間

### 分數計算
```
基礎分 = 剩餘時間 × 10 + 鑰匙數 × 100
最終分 = 基礎分 × 難度倍數
```

難度倍數:
- 簡單: 0.5x
- 正常: 1x
- 困難: 2x
- 地獄: 5x

## 🐛 已知問題

- 多人遊戲需要後端伺服器
- 音效在某些移動設備上受限
- 大屏幕上可能需要調整 Canvas 大小
